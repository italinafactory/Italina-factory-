# Italina Factory — site vitrine premium

Site statique responsive, prêt pour GitHub Pages.

## Installation sur GitHub

1. Décompressez l'archive.
2. Ouvrez le dépôt GitHub du site.
3. Supprimez les anciens fichiers du dépôt si nécessaire.
4. Importez **tout le contenu** de ce dossier à la racine du dépôt.
5. Validez avec **Commit changes**.
6. Ouvrez **Settings → Pages**.
7. Choisissez **Deploy from a branch**, branche `main`, dossier `/ (root)`.
8. Le fichier `CNAME` relie le site à `italinafactory.fr`.

## Contenu

- Animations au défilement
- Hero animé
- Galerie filtrable et agrandissement en lightbox
- Vidéo NFC
- Responsive mobile/tablette
- Métadonnées SEO, Open Graph et données structurées
- Sitemap et robots.txt

Les coordonnées du site se trouvent dans `index.html`.
