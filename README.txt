SITE ITALINA FACTORY — MODE D'EMPLOI

1. Le site est prêt à être hébergé gratuitement sur Cloudflare Pages, GitHub Pages ou Netlify.
2. Le fichier principal est index.html.
3. Pour ajouter tes photos : place-les dans le dossier assets puis remplace les blocs de la section « Réalisations » par des balises image.
4. Pour ajouter ton vrai logo : place le fichier dans assets/logo.png et remplace le bloc .brand dans les pages.
5. Avant publication, complète le nom et l'adresse de l'hébergeur dans mentions-legales.html.
6. Le formulaire de contact ouvre l'application e-mail du visiteur ; il ne nécessite pas de serveur.
